# devr taps

## Available Formulae

### `cleanr`

AI testing SDK and CLI for CI validation.

Install with:

```bash
brew install devr-tools/tap/cleanr
```

### `szr`

Token-aware CLI proxy that trims noisy command output for LLM workflows.

Install with:

```bash
brew install devr-tools/tap/szr
```

### `codeguard`

Repository policy and AI code review CLI for CI.

Install with:

```bash
brew install --HEAD devr-tools/tap/codeguard
```

### `merger`



Install with:

```bash
brew install --HEAD devr-tools/tap/merger
```
